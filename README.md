# Airline Reservation System ✈️

A full-stack MERN application for managing airline bookings and reservations.

## Features
- User authentication & authorization
- Flight search and availability check
- Ticket booking and reservation management
- RESTful APIs with structured backend
- Responsive and user-friendly UI

## Tech Stack
- Frontend: React, CSS / Tailwind
- Backend: Node.js, Express.js
- Database: MongoDB
- Version Control: Git & GitHub

## Project Structure
client/ → Frontend  
server/ → Backend APIs
